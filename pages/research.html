// Thrust 01

Resident Space Object Tracking & Orbit Determination

With over 27,000 tracked objects — and many more untracked — accurate orbit determination is the cornerstone of space safety. We develop and test novel algorithms for initial orbit determination (IOD), orbit propagation, and maneuver detection using sparse, noisy, and heterogeneous observation data.

Key methods include Batch Least Squares, Extended and Unscented Kalman Filters, and Particle Filter approaches robust to non-Gaussian uncertainty. We also investigate the use of angles-only observations from ground-based optical telescopes for deep-space RSO tracking.

// Thrust 02

Multi-Sensor Fusion & Sensor Tasking

📡

No single sensor can monitor the full catalog. Radar provides range and range-rate; optical sensors deliver high-precision angular positions; and laser ranging offers centimeter-level accuracy for select targets. We study how to fuse these complementary datasets optimally.

A companion problem is sensor tasking: given a finite network of sensors with scheduling constraints, which objects should each sensor observe, and when? We approach this as a resource allocation and decision-making problem under uncertainty.

// Thrust 03

Machine Learning for Space Situational Awareness

The scale of the modern orbital catalog demands automated, data-driven approaches. We apply deep learning, Bayesian neural networks, and reinforcement learning to problems in object characterization (size, shape, material), anomalous behavior detection, and conjunction screening.

A particular focus is developing models that are uncertainty-aware — that know when they don't know — which is essential for high-stakes collision avoidance decisions.
